What Is DNS Security?
DNS translates domain names into IP addresses. DNS security protects this process from manipulation, spoofing, and hijacking. Tests based on NIST SP 800-81r3.

How does it work?
The checker queries Cloudflare's DNS-over-HTTPS API to retrieve DNS records and runs 8 security tests. No traffic is sent to your servers.
What is DNSSEC?
DNSSEC adds cryptographic signatures to DNS records. Without it, attackers can forge DNS responses to redirect users to malicious sites.
